# Actividad 04: Manipulación del DOM

Este proyecto es una práctica de manipulación del DOM (Document Object Model) utilizando JavaScript puro. Contiene cuatro herramientas interactivas diseñadas para demostrar el manejo de eventos, selección de elementos y modificación de contenido y estilos en una página web.

## Funcionalidades

### 1. Repetidor de Palabras
Permite ingresar una palabra y un número para repetir dicha palabra la cantidad de veces especificada. Utiliza el método `.repeat()` de los strings de JavaScript.

### 2. Cambio de Color de Fondo
Un selector de color que permite cambiar dinámicamente el color de fondo de un párrafo de ejemplo. Demuestra la manipulación de la propiedad `style` de un elemento.

### 3. Calculadora Básica
Realiza las cuatro operaciones matemáticas básicas (Suma, Resta, Multiplicación y División) entre dos números ingresados. Además, calcula una "Suma Total" de todos los resultados obtenidos.

### 4. Inversor de Texto
Toma una cadena de caracteres y la devuelve invertida (al revés). Implementado mediante la combinación de `.split('')`, `.reverse()` y `.join('')`.

## Cómo usar el proyecto

1. Descarga o clona los archivos `index.html` y `script.js` en una misma carpeta.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Interactúa con las diferentes secciones ingresando datos y haciendo clic en los botones de acción.

## Tecnologías utilizadas

*   **HTML5**: Estructura semántica básica.
*   **JavaScript (Vanilla)**: Lógica de programación y manipulación del DOM.
*   **CSS (Nativo)**: Estilos básicos para la interfaz.
